# 简介

**TIDAS**（**TI**angong **DA**ta **S**ystem）是一个基于 JSON 构建的开源数据系统，为生命周期评估（LCA）和碳足迹管理提供统一、标准化、可拓展的底层支撑。它由“方法论、格式规范、数据资源”三大核心模块组成，并通过辅助工具与集成能力，实现智能化、可信化、自动化的数据管理体验。

## 🧱 系统架构核心模块

TIDAS 架构由以下三部分组成：

### 📖 方法论（Methodology）
-  [LCA / 碳足迹数据库构建导则](https://www.carbonfootprint.network/docs/category/lca-database-guideline)  
  提供数据构建、验证与维护方法，兼容 ISO 14040/44、ILCD Handbook 与 ecoinvent等标准。

### 🧾 数据结构（Format）
-  [TIDAS 数据结构](/docs/category/tidas-schema)  
  基于 JSON 与 JSON Schema 实现数据结构验证，适配 AI、数据空间与区块链等新型架构。

### 📦 数据资源（Data）
-  [基础数据包](https://github.com/linancn/EF-reference-package-3.1-Correction)  
  提供开源的参考数据库，支持数据可追溯性、一致性和可信度。

---

## 🛠 辅助工具（Tools）

**TIDAS 工具包是围绕三大核心模块开发的辅助组件**，便于开发者与系统之间进行数据构建、格式互通与导出操作。

- 🧪 `validate`：数据验证工具 – 检查数据结构、规范性、完整性  
- 🔄 `convert`：格式转换器 – 实现 TIDAS 与 eILCD 双向无损转换  
- 📤 `export`：数据包发布工具 – 将合规数据打包为标准 ZIP 格式，用于平台交付与归档

👉 全部工具说明详见：[TIDAS 工具包](/docs/category/tidas-tools)

## 🔗 集成能力(Features)

TIDAS 已构建多种面向实际场景的集成方案，支持可信计算与数据互操作：

- 🔐 [权限控制（Permission Control）](./integration/tidas-permission-control.mdx)：实现多方协作下的数据访问控制与授权机制  
- 🧠 [隐私计算（Privacy Computing）](./integration/tidas-privacy-computing.mdx)：基于多方安全计算与联邦分析实现数据可用不可见
- 🔗 [区块链集成](./integration/tidas-blockchain.mdx)：实现数据不可篡改与分布式验证  
- 🤖 [AI 集成（MCP 服务）](./integration/tidas-to-ai.mdx)：集成 MCP AI Server，支持数据语义分析、数据补全与智能验证
